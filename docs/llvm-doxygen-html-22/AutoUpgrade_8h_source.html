36 /// This is a more granular function that simply checks an intrinsic function
37 /// for upgrading, and returns true if it requires upgrading. It may return
38 /// null in NewFn if the all calls to the original intrinsic function

52 /// This is an auto-upgrade hook for any old intrinsic function syntaxes
