25/// Splits the given module \p M into parts. Each output part is passed to
26/// \p Callback for further possible processing. Each part corresponds to a
27/// subset of the module that is transitively reachable from some entry point
28/// group. Each entry point group is defined by \p EntryPointCategorizer (EPC)
29/// as follows: 1) If the function is not an entry point, then the Categorizer
30/// returns std::nullopt. Therefore, the function doesn't belong to any group.
31/// However, the function and global objects can still be associated with some
32/// output parts if they are transitively used from some entry points. 2) If the
33/// function belongs to an entry point group, then EPC returns an integer which
34/// is an identifier of the group. If two entry points belong to one group, then
35/// EPC returns the same identifier for both of them.
36///
37/// Let A and B be global objects in the module. The transitive dependency
38/// relation is defined such that: If global object A is used by global object B
39/// in any way (e.g., store, bitcast, phi node, call), then "A" -> "B".
40/// Transitivity is defined such that: If "A" -> "B" and "B" -> "C", then "A" ->
41/// "C". Examples of dependencies:
42/// - Function FA calls function FB
43/// - Function FA uses global variable GA
44/// - Global variable GA references (is initialized with) function FB
45/// - Function FA stores the address of function FB somewhere
46///
47/// The following cases are treated as dependencies between global objects:
48/// 1. Global object A is used by global object B in any way (store,
49/// bitcast, phi node, call, etc.): an "A" -> "B" edge will be added to the
50/// graph;
51/// 2. Function A performs an indirect call of a function with signature S, and
52/// there is a function B with signature S. An "A" -> "B" edge will be added
53/// to the graph;
54///
55/// FIXME: For now, the algorithm assumes no recursion in the input Module. This
56/// will be addressed in the near future.
57LLVM_ABI void splitModuleTransitiveFromEntryPoints(
58 std::unique_ptr<Module> M,
59 function_ref<std::optional<int>(const Function &F)> EntryPointCategorizer,
60 function_ref<void(std::unique_ptr<Module> Part)> Callback);
